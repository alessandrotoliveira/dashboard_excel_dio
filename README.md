📊 Dashboard de Vendas no Excel
📌 Descrição do Projeto

Este projeto tem como objetivo a criação de um dashboard de vendas no Excel, com foco na organização, análise e visualização de dados. A proposta é transformar dados brutos em informações claras e visuais, facilitando a interpretação dos resultados e apoiando a tomada de decisões estratégicas.

🎯 Objetivos
Organizar dados de vendas de forma estruturada
Criar visualizações interativas e intuitivas
Analisar o desempenho de vendas ao longo do tempo
Identificar padrões, tendências e oportunidades de melhoria

🧰 Ferramentas Utilizadas
Microsoft Excel
Tabelas Dinâmicas
Gráficos Dinâmicos
Segmentação de Dados
Formatação Condicional
